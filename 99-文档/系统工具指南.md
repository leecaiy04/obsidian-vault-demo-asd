# 🔧 系统工具指南

这些是默默在后台工作的系统级插件。

## 1. BRAT (Beta Reviewers Auto-update Tester)
**作用**: 安装 GitHub 上的**测试版插件**，或还没上架社区市场的插件。

### 如何安装非市场插件
1.  命令面板搜 `BRAT: Add a beta plugin for testing`。
2.  输入 GitHub 仓库地址 (如 `TfTHacker/obsidian42-brat`)。
3.  点击 `Add Plugin`。
4.  BRAT 会自动帮你下载并保持更新。

> ⚠️ **警告**: Beta 版插件可能不稳定，建议仅用于尝鲜。

---

## 2. Global Proxy (全局代理)
**作用**: 解决 Obsidian 无法连接社区市场、无法下载主题的问题。

### 为什么需要？
在国内网络环境下，访问 GitHub 或 Obsidian 官方 API 经常失败。该插件允许 Obsidian 走你的本地代理（如 Clash/v2ray）。

### 配置
- **Server**: 通常是 `127.0.0.1`。
- **Port**: 查看你代理软件的端口（常见 7890, 10809 等）。
- **Type**: HTTP 或 SOCKS5。

---

## 3. i18n (翻译)
**作用**: 汉化未提供中文语言包的插件。

有些冷门插件只有英文界面，i18n 插件可以加载社区贡献的翻译文件，将其界面汉化。如果你发现某个插件全是英文看不懂，可以尝试用它来查找翻译包。
